# Godot 4 - Top-Down Dash Mechanic
An 8-way movement controller demonstrating Vector math and asynchronous timers.

### Mechanics Demonstrated:
*   `Input.get_vector()` for smooth gamepad/keyboard analog movement.
*   Vector normalization to prevent the "diagonal speed exploit".
*   Asynchronous GDScript coroutines using `await get_tree().create_timer()`.
